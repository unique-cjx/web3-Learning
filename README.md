# Web3-Learning

## What is Web3
> Web3 的核心是通过区块链、加密货币和非同质化代币将权力以所有权的形式归还用户
1. web3是去中心化的
2. web3无需许可
3. web3具有原生支付功能（金融属性）
4. web3无需信任
